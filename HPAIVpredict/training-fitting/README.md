# detailed training and fitting of the model using R


* `COMPLETECORR.v3.html` contains the training details in HTML format, including a plot of the insertions used to train the model and a comparison between predicted and observed insertion frequencies.

* `COMPLETECORR.v3.Rmd` contains the R code used to generate the model. This script requires the additional files `allinsertions.xlsx` and `predictioncodonH5-vmod.xlsx`. To run it, download the R Markdown file together with both Excel files, open the `.Rmd` file in RStudio, and run or knit it.

# Note

* The additional file `predictioncodonH5-vmod.xlsx` contains a subset of the ΔΔG estimates generated by running `inserpredictor.py` on the `allvirus.fasta` example provided in the `HPAIVpredict/` folder. Specifically, it includes predictions for the HA sequences of the Shim-series viruses (Shim, ShimA1, ShimA2, ShimA3, ShimA4, and Shim24a2b) and PR8 (H1N1).

* The second file, `allinsertions.xlsx`, contains the experimentally observed insertions detected using PA-HA and minigenome constructs containing HA sequences from the Shim-series viruses (Shim, ShimA1, ShimA2, ShimA3, ShimA4, and Shim24a2b) and PR8 (H1N1). This file was generated using the `sequencing-analysis/` pipeline, followed by the manual insertion-mapping steps described in the example folder.
