# HA-insertion

This github repository contains all the bioinformatic resources coming from the paper "Insertion-driven emergence of high pathogenicity avian influenza viruses is governed by product–template duplex thermodynamics".

The repository is organized into three sub-repositories:
    • HPAIVpredict: contains the complete set of scripts used to predict polymerase slippage-mediated insertion events. The accompanying example dataset includes all viral sequences analyzed in this study.
    • Sequencing-analysis-H5: contains the complete bioinformatics pipeline used to identify insertion events from high-throughput sequencing data.
    • Stationary-and-Transient-Structure-Analysis: contains the nucleotide sequences, scripts, and detailed instructions required to reproduce the RNA secondary structure analyses presented in this study.
