# HA-insertion

This github repository contains all the resources coming from the paper , 

three internal repositories contain 
**HPAIVpredict**: containing the full scripts to predict slipagge insertions, example data correspond to all the viruses present in the paper;
**Sequencing-analysis-H5**: contains the full pipeline used to detect insertions coming from sequencing data;
**stationary and transient structure analysis**: contains sequences and instruccion to calculate RNA structures analized in the paper.
